# AGENTS.md

## Project

This is a multilingual Next.js website for Eliezer Torjmane Tax Advisor.

## Commands

- Install: `npm install`
- Develop: `npm run dev`
- Lint: `npm run lint`
- Build: `npm run build`

## Content Rules

- French is the primary complete version.
- Hebrew pages must support RTL layout.
- English pages should stay concise and aligned with the French content.
- Keep using `public/logo.png` as the main brand asset.

## Legal Boundaries

- Do not claim Eliezer is a CPA, lawyer, רואה חשבון, or licensed tax consultant.
- Do not guarantee tax refunds.
- Use careful wording such as conseiller fiscal, accompagnement fiscal, orientation, préparation and accompaniment in processes.
- Keep the footer disclaimer visible:

  Les informations présentes sur ce site sont fournies à titre général et ne remplacent pas un conseil personnalisé. Chaque situation fiscale doit être analysée individuellement.

## Design Notes

- Keep the light premium blue-on-white advisory system.
- Use the brand accent colors `#7DDCFF`, `#4FAFD1`, and `#1E6B87`.
- Use white and very light blue backgrounds, clean navy typography, soft borders, subtle shadows and clear blue gradient CTAs.
- Keep UI elements compact, modern, reassuring and mobile-first.
