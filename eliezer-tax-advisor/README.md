# Eliezer Torjmane Tax Advisor

Premium multilingual website for **Eliezer Torjmane Tax Advisor**, a French-speaking tax advisor in Israel.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide icons

## Languages

- French: main complete version at `/fr`
- Hebrew: RTL support at `/he`
- English: `/en`

The root route redirects to `/fr`.
